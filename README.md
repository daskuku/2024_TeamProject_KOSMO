## 프로젝트 주제 
JSP 기반 축구 커뮤니티 사이트

## 개발기간
2024.05.01 ~ 2024.05.24

## 개발목표
* 여러 사이트를 돌아다닐 필요없이 한 곳에서 경기장 대관 시스템 구축
* 팀/팀원, 용병, 레슨 모집 기능을 구현해 사용자의 접근성 향상
* 축구관련 커뮤니티

## 맡은 파트
* 마이페이지
* 모집(팀/팀원, 용병, 레슨)
* 경기 매칭
* 대회 일정

## 사용기술
* Java
* JavaScript
* Oracle
* JQuery
* HTML
